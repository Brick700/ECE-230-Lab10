### What is difference between edge and level sensitive circuits?
Level-sensitive circuit updates the output whever the enable is set to high so the output folows the data at every point. Edge-sensitve circuits only captures the inpjut the moment the clock goes from low to high.
### Why is it important to declare initial state?
It is important to declar intial state as flip-flops start in unknown state, so it could lead to an output with unknowns until it hits the first clock edge or even farther.
### What do edge sensitive circuits let us build?
